# Portfolio Website

A premium, developer-focused portfolio website built with Next.js 14, Tailwind CSS, and Framer Motion.

## Features

- **Responsive Design**: Looks great on all devices (mobile, tablet, desktop).
- **Dark Mode Aesthetic**: Sleek, modern dark theme with electric blue accents.
- **Scroll Animations**: Smooth reveal animations using Framer Motion.
- **Interactive Components**: Hover effects, glassmorphism, and dynamic elements.
- **Typing Effect / Gradients**: Modern typography treatments.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Customization

- **Content**: Update the data in the component files (`app/components/*.tsx`) or move them to a separate `data/` folder.
- **Colors**: Modified in `app/globals.css` under the `@theme` directive.
- **Fonts**: Using `Inter` from `next/font/google`, configured in `app/layout.tsx`.
